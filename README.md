## Instalação do Hemera:

### Kali Linux / Ubuntu / Mint / Debian / Fedora

```
* git clone https://github.com/Diego333-ms/Hemera
* cd Hemera
* chmod +777 linux_install.sh
* ./linux_install.sh
* python3 Hemera.py ou ./Hemera.py
```

## Termux / Android

```
* git clone https://github.com/Diego333-ms/Hemera
* cd Hemera
* chmod +777 termux_install.sh
* ./termux_install.sh
* python3 Hemera.py ou ./Hemera.py
```
